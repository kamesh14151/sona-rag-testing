# Sona RAG Testing Repository

This repository is dedicated to testing the **Sona RAG (Retrieval-Augmented Generation)** chatbot.

## Overview

The Sona RAG system is designed to provide college assistance by leveraging a combination of local knowledge bases and live human agent integration. This ensures that students receive accurate, real-time information.

## Key Features

- **Retrieval-Augmented Generation**: Uses a knowledge base to provide contextually relevant answers.
- **Human-in-the-loop**: Integrates with live agents for complex queries.
- **Easy Integration**: Can be embedded into any website using a simple iframe.
